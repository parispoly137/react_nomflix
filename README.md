### Screens Explanation

- _Home_: 영화들이 보여짐
- _TV show_: TV Route가 존재함. TV show들이 보여짐
- _Search_: TV show나 영화를 찾을 수 있음
- _Detail_: 영화나 TV쇼를 클릭하면 각각의 디테일이 나옴
